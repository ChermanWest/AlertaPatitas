# Coloca aquí tus hojas de estilo

Copia `home.css`, `auth.css` (y cualquier otro `.css` que quieras reutilizar) dentro de esta
carpeta, e impórtalos en `src/main.jsx`:

```js
import './styles/home.css';
import './styles/auth.css';
```

Todas las clases (`.pet-card`, `.auth-input`, `.btn-publish`, `.filters`, etc.) se dejaron
intactas en los componentes para que tu CSS actual siga aplicando sin cambios.
